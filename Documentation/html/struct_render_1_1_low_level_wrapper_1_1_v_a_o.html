Detailed Description

My VertexArrayObject structure to handle 3D models efficiently in VRAM.

Allocate VBOs (buffers) to store vertices, normals and colors of the mesh
Then load mesh attributes to the GPU (VRAM)
Use the ID of the VBO to display from the VRAM
Then free ressources on the GPU!

Allocate VBOs, send buffers to VRAM, so you can use VAO.

You have to specify the vertices, normals, colors, faces and then choose the buffer type (see parameters of mode).

normalsVector of GLfloat must be allocated.
